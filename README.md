# 项目介绍
**题目** 第二十三届江泽涵杯B题：交通网络的健壮性和提升

## 项目结构
- cases: 题目数据，包括处理之后的数据
- figures: 运行程序得到的图片
- run_info: 运行程序得到的数据，按问题分类
- src: 源代码
  - community.py: 对城市地图进行社区划分并可视化
  - graph_visulize.py: 绘制城市地图，支持显示路径
  - plot.py: 简易的绘图程序
  - praph_process.py: 分析输入数据，得到连通分量的数目，大小等特征
  - q1.py: 第一问的内容，主要为数据特征分析，包括度分布，聚类系数，直径等
  - q2.py: 第二问的内容
  - q3.py: 第三问的内容，使用度数/聚类系数作为启发
  - q3_lir.py 第三问的内容，使用二核攻击和进化算法
  - q3-flow.cpp: 第三问的内容，使用网络流模拟进行启发
  - q4.cpp: 第四问的内容，使用网络流模拟进行启发
  - q5-1.cpp: 第五问的内容，分析新增路径，单点破坏的情况
  - q5-2.cpp: 第五问的内容，分析新增路径，范围破坏的情况
  - result_process.py 运行结果可视化
- others: 项目中进行的其他尝试
  - 3_core: 三核攻击性能曲线
  - degree_list: 二核攻击时网络总度数变化
  - eccentricity: 节点与相邻最远节点的距离分布
  - evolution_history: 进化算法训练曲线
  - ga_results: 取进化算法最佳结果进行测试得到的性能曲线
  - q5_demo_plot: 分析城市核心区域及相关特征
  - robustness_line: 删除一定比例节点之后剩余网络的鲁棒性
  - q5_lir.py 第五问探索，暂未完成
